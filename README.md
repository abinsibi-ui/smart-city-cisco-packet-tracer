# Smart City using Cisco Packet Tracer

## Project Overview
This project simulates a Smart City monitoring system using Cisco Packet Tracer IoT devices. The system integrates smart surveillance, intelligent street lighting, and fire monitoring to improve safety and energy efficiency.

## Features
- Smart Surveillance using IP cameras and motion sensors
- Intelligent street lighting based on light sensors
- Fire monitoring using fire sensors and alarm systems
- Centralized monitoring through IoT server

## Technologies Used
- Cisco Packet Tracer
- IoT Devices
- Networking (Router, Switch, Gateway)

## How to Run
1. Download the `.pkt` file
2. Open it in Cisco Packet Tracer
3. Run the simulation
4. Trigger sensors to test the system
